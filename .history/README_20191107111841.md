# Vue Element 管理后台
## 项目运行
#### [在线预览](https://calamus0427.github.io/vue-element-admin-calamus/#/login)
 安装依赖：npm install

 运行：npm run dev

 编译：npm run build


## 技术栈
- vue
- element
- vue-router
- vue-markdown
- echarts
- vue-codemirror
- vue-editor2
- vue-bootstrap
- sass
- mavon editor
- mockjs
- webpack
- axios
